Number Formatter
=========

A small library that adds commas

## Installation

  `npm install @jdaudier/number-formatter`

## Usage

    var numFormatter = require('@jdaudier/number-formatter');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests

  `npm test`
